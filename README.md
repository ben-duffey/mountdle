# Mountdle

A daily game for World of Warcraft mounts. Guess the mystery mount of the day.

## Data sourcing and cleaning
Mount data was sourced from a combination of [Blizzard's API](https://community.developer.battle.net/documentation/world-of-warcraft/game-data-apis) and a fan website, [Warcraft Mounts.](https://www.warcraftmounts.com/)

Some mounts have been omitted from this game, including mounts that are stored in the game files but not available (upcoming mounts in new patches, planned mounts from previous expansion that were never released) as well as some mounts that did not have clean data.

## Development Stack/Tools

- Frontend: vanilla HTML/CSS/JS
- Backend: Node.js + Express
- AI was used for this project. Mount data collection and cleaning was done with the help of Claude Code.
